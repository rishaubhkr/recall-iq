# AI Prompt Customization Plan

## Objective
Make the number of cards generated by the AI customizable on the Bulk Import page. This allows admins to specify exactly how many cards they want in a single batch.

## 1. State Management (`frontend-specialist`)
- Add `cardQuantity` state to `BulkImportPage`, defaulting to 10.
- Update `buildPrompt` to accept `cardQuantity`.

## 2. UI Implementation (`frontend-specialist`)
- In the "AI Master Prompt" modal, add a labeled number input for "Quantity".
- Position it above the prompt preview.
- Ensure it updates the `cardQuantity` state on change.

## 3. Prompt Logic (`frontend-specialist`)
- Update the `buildPrompt` template to use the `cardQuantity` variable: `Generate ${ctx.cardQuantity} diverse cards.`

## 4. Verification (`test-engineer`)
- Run linting.
- Manually verify that changing the quantity input updates the prompt preview and the copied text.

## Execution Strategy
- Sequential update to `src/app/(admin)/admin/import/page.tsx`.
- Add the input field with glassmorphism styling to match the theme.
